# Walnet

**Walnet** is a decentralized platform for sharing **CPU/GPU resources** to enable **high-quality cloud gaming** and **heavy tasks handling**. Built on the **Sui blockchain**, Walnet allows users to play **AAA games** by tapping into the computing power of other devices, eliminating the need for expensive gaming hardware.
